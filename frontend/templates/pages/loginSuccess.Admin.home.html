Tokenization

Tokenization is a very common task in NLP, it is basically a task of chopping a given string into pieces, called as token, and throwing away the certain characters at the same time, like punctuation. Improper tokenisation can lead to subpar performance in downstream tasks. Here user can try few models for tokenisation.

Text Representation

Word Embeddings are a distributed representation for text that allows words with similar meaning to have a similar representation. State of the art embeddings like BERT, Albert, Distillibert, perform very well in NLP tasks. Here user can try to generate these embeddings for a sentence.

Transliteration

ISO 15919 "Transliteration of Devanagari and related Indic scripts into Latin characters" is a standard for romanization which uses diacritics to map the much larger set of consonants and vowels in Brahmic scripts to the Latin script. Transliteration is a common tool in any pipeline used for processing multilingual and code mixed language data.

Named Entity Recognition

NER is probably the first step towards information extraction that seeks to locate and classify named entities in text into pre-defined categories such as the names of persons, organizations, locations, expressions of times, quantities, monetary values, percentages, etc. NER is used in many fields in Natural Language Processing.

Sentimental Analysis

Sentiment Analysis is the process of determining whether a piece of writing is positive, negative or neutral by using NLP and ML techniques. Here the user can try sentiment Analysis for pure english text and hindi-english code mix text.

Summarization

Summarization system consists of reduction of a text document to generate a new form which conveys the key meaning of the contained text. Extractive summarization means identifying important sections of the text and generating them verbatim producing a subset of the sentences from the original text.
