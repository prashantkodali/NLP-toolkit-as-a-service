Tokenization

Description Of Our Tokenizer

A tokenizer is in charge of preparing the inputs for a model. This tokenization service provides three options - 1. Word Tokenization using NLTK (python library) 2. Word Tokenization using Transformers with BERT Model and 3. Sentense Tokenization.

References

HuggingFace's Transformers: State-of-the-art Natural Language Processing, https://arxiv.org/abs/1910.03771

Author

Neha Motlani

neha.motlani@research.iiit.ac.in



API Details:

API end point

http://localhost:8080/gettokenizer

Input JSON format

JSON should consist of one argument :
key: "text" , value: text for which tokens need to be generated

Output JSON format

Service returns a JSOn with one key
key: "outputtext" , value: Tokens of the sentences
