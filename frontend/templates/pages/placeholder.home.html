Project Description

This is the course project Demo site for CSE 461 Software Engineering, at IIIT Hyderabad, for the semester of Spring 2020.

NLP-toolkit-as-a-service, is a portal where developers can check the existing NLP models developed, or tried in an organisation / institution. A developer can publish his work in the portal, where other developers can explore the tool, try sample outputs. Also, information regarding the APIs of these each of these services will be shared by the developers here, which will enable a user to integrate the services into their tools is they wish to.
