Transliteration

Description Of Our Tokenizer

This transliteration service is for hindi in Devanagari script to romanized hindi that employs ISO 15919 standard for romanization. This service uses a character to diacritics map to compute the romanized character for a given character in Devanagari script.

References

ISO-15919 standard wikipedia page

Libraries and Resources used

Code base is is Python. No special libraries or packages are used.

API Details:

API end point

http://localhost:7000/transliterate

Input JSON format

JSON should consist of only one argument:

key: "text" , value: Devanagari script for which romanized text is to be generated.

Output JSON format

Service return a JSON with two keys

key : "transliterated_text", value: Summary of the sentence.

key : "error", value: 'None' if no error, else Error Description.
