Sentiment Analysis

Description of Sentiment Analysis

Sentiment Analysis (SA) is the interpretation and classification of sentiment (positive, negative and neutral) within text data using text analysis techniques. Here, our sentiment analysis service detect polarity score within a given text, based on that polarity score assigns a sentiment label. And, In this toolkit we are building the SA model for normal English text and Hinglish (Hindi and English Code-Mixed data) text. To build the model for nomal English text used the open-source libraries TextBlob and NLTK's VADER (Valence Aware Dictionary and sEntiment Reasoner) and for Code-Mixed text used the MultiLayer Perceptron (MLP) with TFIDF vectors.

References

Pruthwik M., Prathyusha D., Pranav D. (2017). "Code-Mixed Sentiment Analysis Using Machine Learning andNeural Network Approaches." ArXiv abs/1808.03299 (2018): n. pag.

Input JSON format

JSON should consist of two arguments :

key: "text" , value: User input text.

key: "id", value: User have to choose either English Sentient Analysis System, or Code-Mixed System.

Output JSON format

Service return a JSON with only one key

key : "label", value: Sentiment label of the user input.
